# Chat — C++ 多服务登录注册系统（开发中）

一个基于 **C++17 / Qt / CMake / WinSock** 的多服务登录注册系统工程。  
项目采用 **客户端 + 网关服务 + 状态服务 + 聊天服务预留模块** 的模块化架构，当前已完成登录、注册流程、客户端基础界面以及基础通信逻辑，聊天相关功能仍在后续开发中。

---

## 项目简介

本项目主要用于实践 **C++ 网络编程、Qt 客户端开发、多服务架构拆分与工程化组织**。  
当前阶段以 **登录 / 注册系统** 为核心，已完成从客户端界面到服务端请求处理的基础链路搭建，为后续聊天功能扩展提供系统基础。

目前已实现的核心能力包括：

- 用户注册与登录流程
- Qt 客户端基础界面
- 登录后消息输入与发送界面
- 网关服务请求接入与分发
- 状态服务基础框架
- Socket 通信基础封装
- 基于 CMake 的多模块工程组织
