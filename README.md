# 🏷️ Chat — C++ 多服务聊天系统（开发中）

这是一个基于 **C++ / Qt / CMake / WinSock** 的多服务聊天系统工程，  
当前处于 **系统前期与基础功能实现阶段**。

> ⚠️ **项目尚未完成完整的实时聊天功能**  
> 目前已实现：**登录 / 注册流程 + 消息发送界面与基础通信逻辑**  
> 实际聊天消息的完整分发与同步仍在开发中。

---

## 📌 当前项目状态（重要）

### ✅ 已实现功能

- 用户 **注册 / 登录** 流程
- 客户端基础界面（Qt）
- 登录后 **消息输入与发送界面**
- 多服务基础架构搭建
- Socket 通信基础封装
- 服务端模块拆分（网关 / 状态 / 聊天）

### 🚧 尚未完成 / 规划中

- 实时聊天消息的完整转发
- 聊天消息的持久化存储
- 多用户并发聊天
- 消息历史同步
- 完整协议设计与异常处理

---

## 🧠 模块说明（当前实现情况）

### 🔐 GateServer（已实现基础功能）

- 处理客户端登录 / 注册请求
- 校验用户信息
- 将客户端请求分发到后端服务

---

### 📊 StatusServer（基础完成）

- 维护用户在线状态
- 为后续聊天功能提供状态支持
- 当前以服务框架与接口为主

---

### 💬 ChatServer（开发中）

- 预留聊天消息处理接口
- 基础 socket 通信已完成
- 实际消息广播与同步逻辑 **尚未完成**

---

### 🖥️ life（客户端）

- Qt 实现的客户端界面
- 支持：
  - 登录
  - 注册
  - 登录后消息输入与发送界面
- 当前用于验证系统前期流程
